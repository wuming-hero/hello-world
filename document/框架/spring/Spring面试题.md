# spring循环依赖再怎么解决
1. 生成代理对象产生的循环依赖
这类循环依赖问题解决方法很多，主要有：

* 使用@Lazy注解，延迟加载
* 使用@DependsOn注解，指定加载先后关系
* 修改文件名称，改变循环依赖类的加载顺序

2. 使用@DependsOn产生的循环依赖
这类循环依赖问题要找到@DependsOn注解循环依赖的地方，迫使它不循环依赖就可以解决问题。

3. 多例循环依赖
这类循环依赖问题可以通过把bean改成单例的解决。

4. 构造器循环依赖
这类循环依赖问题可以通过使用@Lazy注解解决。
