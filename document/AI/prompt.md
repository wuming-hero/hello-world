skills市场推荐
* skills.sh：目前活跃度最高的枢纽之一。有实时排行榜和安装量数据，方便发现热门趋势。
* skillsmp.com：当前规模最大的聚合市场之一，支持语义搜索和中文界面，方便进行多维度检索。
* skills.homes：专注于从 GitHub 聚合 Skills。其特色是按“开发工具”、“数据 AI”等实际用途进行智能分类，查找效率高。
* agentskill.sh：特色是为每个技能提供安全评分和审计详情，帮你判断脚本是否存在风险。
* github.com：最权威的来源。例如 Anthropic 官方提供的示例技能包，以及 Vercel Labs 维护的最佳实践集合，是学习高质量 Skill 编写的首选。
* clawhub.ai：帮你快速找到适合 Codex / OpenClaw / Claude Code 的 Skills，提高开发和办公效率。
